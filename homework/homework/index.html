What does HTML stand for?

HTML stands for HyperText Markup Language. The hyperText part refers to the fact that HTML allows you to create links that allow visitors to move from one page to another quickly and easily.

What is semantic markup?

Semantic markup is extra information that is not intened to affect the structure of your web page. Sematic means meaning. Sematic mark up pay attention to the intended meaning of each piece of your work.

How many types of headlines are there?

There are six headlines. Ranging from h1-h6. H1 is used for the main headings, h2 is used for subheadings and it continues down, each getting smaller, all the way down to h6.

What happens if you use multiple spaces?

When in a browser, there are two or more spaces, it only displays it as once space. The same thing happens if there is a line break, it will still treat that space as a single space. This is known as a white space collapsing.
